## Conventions
- Interface et textes utilisateur en FRANÇAIS.
- Code lisible et commenté avec mesure ; noms de variables en anglais.
- Chaque fonctionnalité livrée avec ses tests quand un framework de test existe.
- Mettre à jour ce CLAUDE.md quand une décision structurante est prise
  (stack, architecture, modèle de données) — section « Décisions » ci-dessous.

## Workflow (rappel du contrat des sessions autonomes)
- Les tâches viennent de TODO.md (À faire → En cours → Terminé).
- Une branche feature/<description> + une pull request par tâche. Jamais de code sur main.
- Ne jamais merger. Ne jamais force-push. Jamais de secrets dans le code.
