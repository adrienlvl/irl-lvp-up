# Boucle #195 (autonome, phase 2) — Prochain palier de série · build 1.9.129

**Phase 2 (zone gamification).** La série de jours (🔥) affichait juste le compteur — rien pour donner envie de tenir jusqu'au prochain cap.

## Livré

Sous le compteur de série, un **objectif de palier** apparaît quand on s'en approche (≤ 5 jours) : ex. **« 🎯 +2 → 7 »** (encore 2 jours pour atteindre 7 jours d'affilée), en vert accent. Le bloc série a aussi une **infobulle** permanente (« Prochain palier : 30 jours d'affilée (plus que 4) »). Paliers : 3 · 7 · 14 · 30 · 60 · 100 · 180 · 365. N'affiche rien une fois le dernier palier atteint.

## Détail technique

- **`lib/logic.js`** : `STREAK_MILESTONES` + `nextStreakMilestone(current)` → `{milestone, remaining}` ou `null`. Pur + testé.
- **`app.js`** : rendu du `#streakHint` (visible si `remaining ≤ 5`) + `title` sur `#streakBlock`.
- **`index.html`** : `#streakBlock`, `#streakHint`. **`style.css`** : `.streak #streakHint` (accent, masqué si vide).
